## 4. 完了の定義（Definition of Done）— 全エージェント必須
コードを変更したら、「終わった」と言う前に、このリポジトリの検査を通して**緑**にすること:
- **Python**: `ruff check .` ＋ `ruff format --check .` ＋ `pytest`
- **Swift** : `xcodebuild ... test`（または `swift test`）／`swiftlint`
- **JS/TS** : `npm run lint` ＋ `npm test`（または `vitest`）

赤が出たまま完了報告しない。これが「どんなAIが書いても間違いを減らす」仕組み。

## 5. コーディング規約
- その言語の標準フォーマッタ/リンタに従う（手で整形しない）。

- 例外を握りつぶさない。秘密情報（`.env`・APIキー）は**絶対にコミットしない**。
- 返答・説明・コメントは**日本語**。
