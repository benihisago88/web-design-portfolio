# KENCIH — Web Design Portfolio

Web designer / web director portfolio featuring five self-initiated concept projects.

## Case studies

1. Flowbase — B2B SaaS product website
2. NAGI — Boutique hotel booking experience
3. KISSA 04 — Specialty coffee e-commerce
4. MICHI Clinic — Accessible healthcare website
5. MACHI/MA — City design festival platform

All projects are fictional concepts created to demonstrate strategy, information architecture, UX, UI, art direction, and web direction. Target metrics shown in the case studies are design hypotheses, not actual client results.

## Development

```bash
npm install
npm run dev
```

Production build:

```bash
npm run build
```

## Internationalization

English is the default language for international clients. The complete interface and all five case studies are available in English and Japanese.

- Shareable language URLs: `?lang=en` and `?lang=ja`
- Shareable case-study URLs: `?lang=en&case=flow`
- Language choice persists locally when storage is available
- The current case and scroll context are preserved when switching languages
- Dynamic `lang`, page title, description, Open Graph locale, canonical URL, and `hreflang` alternates
- Accessible 44px language controls on mobile and desktop
